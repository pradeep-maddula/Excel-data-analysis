# Excel Data Analysis

📊 Project Overview

This project focuses on analyzing data using Microsoft Excel. It includes data cleaning, transformation, and visualization to extract meaningful insights.

🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Data Cleaning Techniques

📈 Key Tasks

* Cleaned raw data
* Performed exploratory data analysis (EDA)
* Created dashboards and visualizations
* Derived insights from data

📌 Results

* Identified trends and patterns
* Generated actionable insights
